EUD-5701 — Unblock preserves nonce

1. Open DevTools → Console.
2. Click Accept on the banner.
3. A green [UNBLOCKED OK] log should appear. Run window.__unblockedAt — it must be a timestamp string.
4. There must be no Refused to execute inline script.
5. In Elements verify the new <script type="application/javascript"> that replaced the blocked one — it must carry nonce="abc123xyz".
